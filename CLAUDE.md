# crowdworks-agent

CrowdWorks の案件検索・提案文生成・応募フォーム入力を Cursor Agent + browser MCP で自動化するスキル。

## 機能

- **Phase 1**: キーワード/カテゴリで案件を一括検索し、10件をブラウザタブで同時展開
- **Phase 2**: ユーザーが選んだ案件に対して提案文を自動生成し、応募フォームに入力（送信はユーザーが手動で行う）

## セットアップ

### 1. リポジトリをクローン

```bash
git clone <リポジトリURL>
cd crowdworks-agent
```

### 2. Cursor でこのディレクトリを開く

```bash
cursor .
```

`.cursor/skills/crowdworks/SKILL.md` が Cursor Agent のスキルとして自動認識される。追加のインストールや依存関係は不要。

### 3. browser MCP の有効化を確認

Cursor Settings → MCP → `cursor-ide-browser` が **enabled** になっていることを確認する。

これがないとブラウザ操作（案件ページの表示・フォーム入力）ができない。

### 4. 使い始める

Cursor の Agent モード（Cmd+L）で以下のように入力する:

- 「クラウドワークスで AI 案件を10件探して」
- 「CW で案件探して」

スキルが自動トリガーされ、検索 → タブ展開 → 応募ループが開始される。

## CrowdWorks ログインについて

- **案件の閲覧にログインは不要**（`/public/jobs/{id}` は公開ページ）
- **応募時にのみログインが必要**。エージェントが応募フォームに進む際、未ログインであれば「ブラウザでログインしてください」と案内する
- ブラウザで1つのタブからログインすれば、Cookie が共有されて全タブに適用される
- セキュリティ上、エージェントが自動ログインすることはない

## カスタマイズ

このスキルを自分用に使うには、`.cursor/skills/crowdworks/SKILL.md` 内の以下を書き換える:

### プロフィール（Step 2-3）

```
**佐藤のプロフィール:**
- 業務自動化・AI活用のコンサルタントとして独立して活動
- 得意分野: GAS, API連携, ChatGPT/Claude活用, 業務自動化, DX支援
...
```

→ 自分の名前・経歴・得意分野に変更する。

### フィルタキーワード（Step 1-2）

```
- スキップワード: イラスト, デザイナー, ロゴ, ...
- マッチキーワード: AI, GAS, ChatGPT, LLM, ...
```

→ 自分のスキルセットに合わせてキーワードを調整する。

### 提案文テンプレート（Step 2-3）

提案文の構成（自己紹介・提案概要・実装アプローチ・スケジュール・強み）を自分のスタイルに合わせて編集する。
